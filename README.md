# GA SEI Project 3 - Save the Muffin

## Introduction

A MERN app that allows you to purchase unsold food items for value and do your bid to reduce food wastage.

## Technology

**Front-end:**

- HTML
- CSS (using Tailwinds CSS)
- JavaScript
- React

**Back-end:**

- Express
- MongoDB (using Mongoose)
